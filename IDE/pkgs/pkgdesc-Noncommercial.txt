Multitarget Development System by Moravia MIcrosystems, s.r.o.

This is NON-COMMERCIAL variant (cannot be legally used by certain profit making corporations, etc.), read license terms
for more information.

MDS is a window-based software development platform that combines robust and modern editor with a project manager and
make facility tool, integrating all the tools needed to develop embedded applications including macro assembler, and
a HEX, VHDL, Verilog file generator. MDS helps expedite the development process of embedded applications.
