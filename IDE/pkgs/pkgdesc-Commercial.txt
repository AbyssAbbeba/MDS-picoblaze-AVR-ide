MDS - Multitarget Development System by Moravia Microsystems, s.r.o.

This is commercial variant, please read license terms for more information. Congratulations on your buy!

MDS is a window-based software development platform that combines robust and modern editor with a project manager,
integrating all the tools needed to develop embedded applications including macro assembler, and HEX, VHDL, and Verilog
file generator. MDS helps expedite the development process of embedded applications.
